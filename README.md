# 使用GitBook 制作在线文档

markdown + git + gitbook 制作在线文档。

***

## 为什么用GitBook

> gitbook 是通过git的方式构建电子书的一种工具，可以有效的利用了git在版本控制方面的优势和markdown在编写文档方面的优势，再结合github进行在线托管，简直完美。

优点 ：
* 支持markdown语法
* 版本控制
* 可以转换成pdf epub格式


## 安装步骤

* 1、安装node

* 2、安装gitbook客户端


```
npm install gitbook-cli -g
```

* 3、进入https://calibre-ebook.com/download 下载calibre，添加配置calibre安装路径（如C:\Program Files\Calibre2）到 path。
